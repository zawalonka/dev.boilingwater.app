# Wiki Folder Structure

```
wiki/
├── .github/                     # Wiki-specific Copilot instructions
├── docs/                        # Wiki documentation
├── src/                         # Wiki generator scripts
├── dist/                        # Generated static site (ignored in git)
└── .cache/                      # Generator cache (ignored in git)
```

**Note:** The wiki generator only reads game data. It does not modify gameplay code.
