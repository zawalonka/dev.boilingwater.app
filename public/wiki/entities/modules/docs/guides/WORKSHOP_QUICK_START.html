# Quick Workshop Creation Guide

**For creating new Workshops quickly without reading all documentation.**

## 5-Minute Quickstart

### 1. Create Workshop Folder & File

Create folder: `public/assets/workshops/[yourWorkshopName]/`
Create file: `public/assets/workshops/[yourWorkshopName]/workshop.json`

```json
{
  "id": "your-workshop-id",
  "name": "Your Workshop Name",
  "scope": "workshop",
  "description": "Brief description",
  
  "colors": {
    "header_background": "#000000",
    "header_text": "#ffffff",
    "badge_background": "#ff0000",
    "button_primary": "#4CAF50",
    "button_hover": "#45a049",
    "panel_background": "#ffffff",
    "panel_text": "#333333",
    "panel_border": "#cccccc"
  },
  
  "images": {
    "background": "/assets/workshops/your-workshop/background.png",
    "pot_empty": "/assets/workshops/your-workshop/pot-empty.png",
    "pot_full": "/assets/workshops/your-workshop/pot-full.png",
    "flame": "/assets/workshops/your-workshop/flame.png"
  },
  
  "metadata": {
    "author": "Your Name",
    "version": "1.0.0",
    "createdDate": "2026-01-24"
  }
}
```

### 2. Create Asset Folder

Create: `public/assets/workshops/your-workshop/`

Place your 4 images here:
- **background.png** (MUST be exactly **1280√ó800 pixels**, no stretching)
- **pot-empty.png** (transparent PNG, any size)
- **pot-full.png** (transparent PNG, same size as pot-empty)
- **flame.png** (transparent PNG, ~60√ó100 pixels)

### 3. Test

Build and run: `npm run build && npm run preview`

That's it! Your workshop is loaded automatically.

## Image Resizing (1-Liner)

If you have an original image you need to resize to 1280√ó800:

### Using ImageMagick
```bash
magick convert input.png -resize 1280x800! output.png
```

### Using Python
```python
from PIL import Image
img = Image.open('input.png')
img = img.resize((1280, 800))
img.save('output.png')
```

### Online Tool
[Bigjpg.com](https://bigjpg.com) or [Photopea.com](https://www.photopea.com)

## Workshop vs Level Workshops

### Workshop Workshop (applies to ALL levels in a workshop)
```json
{
  "id": "workshop_chemistry",
  "name": "Chemistry Lab",
  "scope": "workshop",
  "workshopId": "workshop_chem_101"
}
```

### Level-Specific Workshop (override for ONE level)
```json
{
  "id": "workshop_chem_adv",
  "name": "Advanced Pressure Cooker",
  "scope": "level",
  "workshopId": "workshop_chem_101",
  "levelId": "level_5",
  "parentWorkshop": "workshop_chemistry"
}
```

## Common Mistakes

‚ùå **Mistake:** Background image is 1024√ó1024
```
‚úì Fix: Resize to exactly 1280√ó800
```

‚ùå **Mistake:** Colors won't apply in the game
```
‚úì Fix: Check JSON syntax is valid (use jsonlint.com)
‚úì Fix: Verify colors are valid CSS values (#fff, rgba(...), etc.)
```

‚ùå **Mistake:** Images show broken link icon
```
‚úì Fix: Ensure file paths in JSON match actual file locations
‚úì Fix: Paths should be relative to public/ folder (/assets/...)
```

‚ùå **Mistake:** Workshop doesn't load
```
‚úì Fix: Check browser console (F12) for error messages
‚úì Fix: Verify JSON file is in public/assets/workshops/[id]/workshop.json
‚úì Fix: Make sure all required fields are present
```

## File Checklist

Before deploying Your workshop:

- [ ] JSON file created: `public/assets/workshops/[id]/workshop.json`
- [ ] All required JSON fields present (id, name, colors, images, metadata)
- [ ] Background image: exactly **1280√ó800 pixels**
- [ ] Background file exists: `public/assets/workshops/[id]/background.png`
- [ ] Pot empty image exists and is transparent PNG
- [ ] Pot full image exists, same size as pot empty
- [ ] Flame image exists (~60√ó100 pixels)
- [ ] JSON validates at [jsonlint.com](https://jsonlint.com/)
- [ ] All color values are valid CSS colors
- [ ] All image paths are correct and accessible
- [ ] Workshop builds without errors: `npm run build`
