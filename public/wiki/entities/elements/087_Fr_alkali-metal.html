Educational notes

Francium is rarest element - only ~1 atom in 10 septillion exists naturally! Most radioactive naturally occurring element. Francium-223 exists briefly before decaying. Never isolated (too radioactive - would kill experimenter). Properties predicted but unmeasured (chemistry inference only). Named after France. Discovered in 1939. Every francium atom exists <20 minutes. Only synthesized in particle accelerators for research.
