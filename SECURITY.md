# Security Policy

## Supported Versions
This project is in pre-alpha and only the latest `main` branch is supported.
Both deployments are in scope for security reports:
- https://dev.boilingwater.app (testing)
- https://boilingwater.app (production)

## Reporting a Vulnerability
Please report security issues privately by email:
**security@boilingwater.app**

If possible, specify which environment is affected:
- dev.boilingwater.app
- boilingwater.app

Include:
- A clear description of the issue
- Steps to reproduce (if possible)
- Potential impact

## Response Expectations
We will acknowledge receipt within 7 days and provide a remediation plan or status update as soon as feasible.

## Disclosure
Please do not publicly disclose vulnerabilities until a fix is available.
